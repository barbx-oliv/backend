# Levantamento de Requisitos 

## Requisitos Funcionais de Não Funcionais 
- Conforme a atividade de Aprendizagem que pedia a realização de um Projeto de Sistema de Gerenciamento de Estoque e Ativos Patrimoniais para uma Unidade Escolar do SENAI, a tabela abaixo é o levantamento de requisitos funcionais e não funcionais.

| ID | Tipo | Descrição |
| ---|  --- |    ---    |
| RF01 | Funcional | Cadastrar funcionários |
| RF01 | Funcional | Permitir que os usuários possam adicionar novos materiais, registrar movimentações de estoque, visualizar inventários e gerenciar ativos patrimoniais |
| RF01 | Funcional | Registro do número de patrimonio de cada item |
| RF01 | Funcional | Registrar saída de materiais e sua devolução |
| RNF01 | Não-Funcional | O sistema deve seguir o padrão de cor da Unidade Escolar |
| RNF01 | Não-Funcional | Apenas usuários registrados podem acessar o sistema |
| RNF01 | Não-Funcional | Senhas e NIF dos usuários devem ser armazenadas |
| RNF01 | Não-Funcional | O sistema precisa estar constantemente ativo para acessibilidade aos usuários |
